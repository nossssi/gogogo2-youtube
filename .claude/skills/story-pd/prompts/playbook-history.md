# playbook 개정 이력·근거 — 평소 로드 금지 (개정·벤치마킹 때만)

`script-guide.md`(공식)의 버전 관리 문서. 집필 시에는 읽지 않는다.

## 개정 문턱 — 이걸 넘어야 개정 후보가 된다

- **서로 다른 채널 2곳 이상에서 반복 관찰**, 또는 **자기 채널 업로드 실측**(조회수·시청 지속)으로 확인된 패턴만.
- 한 영상에서 본 패턴은 노이즈로 간주 — "보류" 표기로 proposal에만 남긴다.
- 벤치마킹 산출물은 **개정 제안까지**. 실제 반영은 사용자 승인 시에만, 반영하면 버전 +1 하고 여기에 기록.
- 대부분의 벤치마킹은 "현행 공식 유효 확인"으로 끝나는 것이 정상이다.

## v2 — 2026-07 (현행)

**변경**: 잘 터진 야담 장편 5편 전문 딥 벤치마킹으로 공식 전면 재구축 + 장편 운영 층(§6) 추가 + 파일 4분할(공식/제약/이력/벤치마킹, 단계별 lazy-load).

**근거 — 벤치마크 5편** (조회 10만+, 전부 88~143분 장편): 뚱이 / 송림 / 황도 / 조선야담꾼 / 손녀야담. 규칙별 출처:
- 콜드 오픈 3요소(충격 장면→결말 흘리기→질문 봉인) ← 5편 공통 오프닝 구조.
- 되감기 브릿지 + 채널 루틴 ← 5편 전부 존재.
- "~아직 알지 못했습니다" 후렴 5~6회 ← 5편 모두 반복한 시그니처.
- 통쾌 주축 + 눈물 양념, 순수 공포 없음(미끼바꿈) ← 5편 감정축 분포.
- 황금률(주인공 직접 복수 금지, 외부 힘이 응징) ← 5편 공통.
- 보상 연쇄 3~5연타 + 주제문 봉인 ← 5편 공통 엔딩.
- 토씨까지 같은 묘사로 복선 회수 / 이중 반전 ← 다수 편 관찰.
- 구연체 2박자(습니다/지요), 시대 특정·지명 모호, 제도 어휘 앵커 ← 5편 문체 공통.
- 장편 층 분량(3~4만자=125~165분) ← 벤치 5편이 전부 88~143분, 채널 방향 결정(2026-07-11).
- **미검증**: 장 분할 수치(10~14장×2,500~3,500자), 클리프행어 로테이션, 배치 순차 집필 — 집필 운영 편의로 설계한 것. **첫 장편 업로드 실측 후 재검증할 것.**
