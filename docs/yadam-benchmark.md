# 야담 흥행 대본 벤치마크 (2026-07 분석)

잘 터진 야담 유튜브 5편(각 조회 10만+)의 전문 자막을 분석해 공통 구조를 역추출한 원자료.
`.claude/skills/story-pd/prompts/script-guide.md`가 이 분석에서 나왔다. 가이드를 개정할 때 근거로 참조.

## 공통 골격 (5편 전원 일치)

1. **콜드 오픈** — 채널 인사보다 먼저 클라이맥스 장면/충격 대사를 시간순 무시하고 맨 앞에. 결말 방향만 흘리고 정체는 감춤. "과연 ~일까요?" 질문으로 닫음. 약 1분/10~19문장.
2. **브릿지** — 채널 루틴("댓글에 건강 남겨주세요" + "지금 바로 시작합니다") → 시간·장소 되감기("조선 숙종 이십오 년…").
3. **8막 아크** — 결핍 → 사건/결단 → 조력자+떡밥 → 갈등 계단식 상승(압박 주체 점층) → 최저점/파국 → 재기+복선회수 → 대반전/응보 → 다중 보상 연쇄+교훈 봉인.
4. **황금률** — 주인공은 직접 복수하지 않는다. 하늘/관아/악인의 자멸이 응징, 주인공은 도덕적으로 깨끗.
5. **순수 공포 없음** — "귀신 폐가"조차 정체는 유산(감동). 괴담은 낚시용 미끼, 실체는 통쾌·눈물.

## 리텐션 장치 (반복 관찰)

- "~는 아직 알지 못했습니다" 막끝 클리프행어 후렴 — 5편 모두 5~6회.
- 떡밥 반복 리마인드(핵심 미스터리를 잊을 만하면 재호출, 같은 문장 재사용).
- "바로 그때였습니다" 전환 트리거.
- 시점 교차 — 흥하는 주인공 ↔ 망하는 악인 cross-cut.
- 테마 대사 3~5회 반향(유훈/격언, 감정 각인+복선 겸용).
- 패턴 인터럽트(장편) — "잠시 숨 고르고 가실까요" 쉬어가기+다음 예고.

## 복선 규칙

- 콜드 오픈 미끼는 반드시 회수.
- 복선은 초반에 촘촘히, 회수 시 토씨까지 같은 묘사 재사용("아, 그거!").
- 이중 반전 흔함(믿었던 권위자=흑막 / 조력자=조상 혼령).

## 문체

- "~습니다" + "~지요/것이었습니다" 구연체 교대. 문어체("~하였다/허나") 금지.
- 감정은 사물·행동 디테일로, 직접 서술 금지.
- 대사 간접화법 위주, 결정적 순간만 직접 인용(짧게).
- 시대 특정("숙종 이십오 년"), 지명 모호("어느 고을"). 조선 제도·경제 어휘로 사실감(환곡·관기·군마·좌수·젓갈·유배).

## 개별 특징 (차용 아이디어)

- **동물/조력자 이중 배치**(송림: 늙은 소가 추격 따돌리기+은닉처 발각 2회 활용).
- **어린아이 해결사**(조선야담꾼: 8살 딸이 가짜 장부 함정 설계·집행).
- **결핍형 떡밥**(손녀: "안 쓰는 오백 냥"이 미스터리·도덕성·증거를 겸함).
- **인포머셜 내장**(황도: 이야기 속 모과차 → 후반 한약사 등장해 실제 효능 강의, 중장년 건강 타깃).
- **경제·상업 성장 서사**(조선야담꾼·손녀: 목장/장시 경영 성취를 리텐션 보상으로).
- **보상 연쇄 극대화**(뚱이: 재앙 격퇴 후 벼락부자→분배→정체밝힘→임금 포상 5연타).

## 자막 정제 주의

자동자막이라 인물명·수치가 흔들림(옥분→옥부/녹분, 천 마리→첫 마리, 보경→복경/오경, 사통→예시/예사). 재작성 시 표준 표기 고정 필요.
